1. установка node.js: https://nodejs.org/en/download/
после установки запустить
npm -v
node -v
Если не запустилось - набрать в поиске винды "environment" и добавить пользовательскую переменную:
"ndeJsPath" = "C:\Program Files\nodejs"
2. Нужно установить тайпскрипт: https://www.npmjs.com/package/typescript
выполняем команду: npm install -g typescript


3.команда на билд:
npm run build
4.команда на старт
npm run start
